== README

This Simple e-commerce app build for Small Business Owner to quickly make a profit.

Feature:

User: Admin can do the basic functionality of small shop in real world: Add Product, manage order, manage inventory, etc
User: Buyer can order and pay with credit card

Tech Stack:

Ruby 2.2.3
Rails 4.2.4
Minitest
Guard
Haml
bootstrap
puma
devise
country_select



<tt>rake doc:app</tt>.
